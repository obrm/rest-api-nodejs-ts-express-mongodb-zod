# REST API with Node.js, Mongoose & TypeScript

## Concepts

- REST API principals
  - CRUD
  - HTTP methods
- JWT & refresh tokens
- Request validation

## Technologies

- Node.js
- MongoDB with Mongoose
- TypeScript
- Express.js & Express.js middleware
- Zod validation

## Features

- logger
- Validate request middleware
- Registration
  1.  User model
  2.  User endpoint
  3.  User session
  4.  Deserialize user middleware (refresh tokens)
  5.  Get sessions
  6.  Delete session
  7.  Require user middleware
- Product resource
  1.  Product model
  2.  Create product
  3.  Read product
  4.  Update product
  5.  Delete product
